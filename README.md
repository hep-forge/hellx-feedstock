Feedstock for [hellx](https://www.roma1.infn.it/~bonvini/hell) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

High Energy Large Logarithms, a small-x resummation code

## Install

```bash
conda install -c hep-forge -c conda-forge hellx
```
